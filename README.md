# bmr_calculator

Basal Metabolic Rate (BMR) Calculator

## Getting Started

Aplikasi BMR Calculator membantu pengguna menghitung Basal Metabolic Rate (BMR) â€” jumlah kalori yang dibutuhkan tubuh untuk menjalankan fungsi dasar seperti bernapas dan sirkulasi darah saat istirahat.
Pengguna cukup memasukkan usia, jenis kelamin, tinggi badan, dan berat badan, lalu aplikasi akan menampilkan hasil perhitungan BMR beserta kebutuhan kalori harian berdasarkan tingkat aktivitas.
Aplikasi ini juga dilengkapi dengan mode terang dan gelap agar nyaman digunakan kapan saja.
